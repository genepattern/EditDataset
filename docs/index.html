Introduction

Edit Dataset (based on the GParc module removeSamples by Barbara Weir) is a GenePattern module that removes user-defined samples from a dataset. The user inputs their dataset as a GCT file, along with a text file containing the list of samples to be removed. The module outputs a GCT file containing the dataset with the user-defined samples removed.

Input Files

  1. GCT File
    Data file containing samples to be removed in a GCT file format.
  2. Remove List
    Text file containing the list of samples to remove. Each sample should be on a separate line and must match a sample name in the file exactly.

Output Files

  1. Data with samples removed [extension].gct
    A GCT file containing the user-inputted dataset with the samples in the removelist text file removed from the dataset.
